# Kore Assistant

An Intellij IDEA extension providing powerful tools for working with [Kore](https://kore.ayfri.com), a Kotlin library for creating Minecraft datapacks without writing JSON.

<!-- Plugin description -->
**Kore Assistant** is a plugin for IntelliJ IDEA that provides a range of features to enhance your development experience with Kore, a Kotlin library for creating Minecraft datapacks.
